JavaScript Language

Parentheses ()

Parentheses, or the grouping operator, are used to:

  • Pass informtaion into a function
    eg. function dostuff (par1, par2) or function dostuff ()
  • Surround conditional statements like if and else/if
    eg. if (x < 10)
  • Surround the statements in a loop
    eg. for (x = 0; x < array.length; x++)
  • Group to enforce order of operations
    eg. a = (4 + 3) * 2;

Brackets []

Square brackets are typically used for defining and accessing the elements of an array or to access properties of objects (instead of dot notation).
eg. var myDog = ["staffy", 5, "tan"] shows the elements in the array myDog
myDog[3] fetches the fourth element in the array myDog

var property_name = 'name';
console.log(myObject[property_name]) logs 'Gabe'


Braces {}

Curly Braces {} are used during the declaration of objects, or to enclose blocks of code (functions, conditionals, loops, classes etc).
eg.
function addOne (n) {
return n + 1
}


Single quotes ' ' and Double quotes " "

Both single quotes and double quotes can be used to define a string, which is a series of characters eg. "car" or "Bronwyn Cotter". Using one over the other is more just a preference and can depend on what is easier to read in your code, and suits the string, especially for words that already have a apostrophe (') or need actual quote marks to be shown around certain text.
eg.
alert("It's game time"); Double works here due to the apostrphe in It's

alert('It\'s "game" time');Single works here due to game being in quotation marks. A backslash is also needed to show the apostrophe is not the end of the single quotation marks.
